# Project Title
 Nova University Chatbot (Rule-Based AI Assistant)

# Description
This project is a simple rule-based chatbot developed using Python.
It helps users get information about Nova University including:
- Courses
- Admissions
- Fees
- Location

# Features
It  Handles 5 user intents which are
- Greeting
- Courses
- Admission (with contact info)
- Fees
- Location
- It gives Human-like responses
- Continuous conversation using a loop
- Exit option (`bye` / `exit`)
- Fallback response for unknown inputs

# Python Libraries Used
- Python Standard Library
- Built-in functions: `input()`, `print()`
- String methods
